边框内圆角

发表于 2016-12-08   |   分类于 CSS

边框内圆角就是只在盒子模型的内侧有圆角，而边框或者描边的四个角在外部仍然保持直角的形状。这个效果我们也有两种方式实现它。

双层 div 解决方案

使用双层 div 实现这个效果比较容易，只需为内层 div 设置圆角效果，然后给外层 div 设置背景色和一定的 padding 即可。
